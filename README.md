## 機能
- [x] 緊急地震速報などの地震情報の通知
- [x] 強震モニタの表示
- [ ] 緊急地震速報の情報をマップ上に表示
- [ ] 地震履歴
- [ ] 地震情報の読み上げ(バックグラウンド/フォアグラウンド)
